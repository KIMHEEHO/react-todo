# React Todo App

React와 MUI(Material UI)를 활용하여 만든 Todo App입니다.  
할 일 추가, 수정, 삭제 기능을 구현하였으며, React Hook과 컴포넌트 구조 분리를 학습하기 위해 제작하였습니다.

## 사용 기술
- React
- Vite
- JavaScript
- MUI(Material UI)
- Tailwind CSS

## React Hook
- useState
- useRef
- useEffect

## 구현 기능
- [X] 할 일 추가
- [X] 할 일 수정
- [X] 할 일 삭제
- [X] 입력 포커스 처리
- [X] 초기 Todo 데이터 생성
- [ ] MUI 기반 UI 적용
- [ ] Tailwind CSS 스타일 적용

## 학습 내용
- 컴포넌트 분리
- Custom Hook(useTodosState) 활용
- 상태 관리
- React Hook 사용
- MUI 컴포넌트 적용
- Tailwind CSS를 활용한 스타일링
